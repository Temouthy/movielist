# MovieList App

Movie list app, where you can add and delete your favorite movies.

STILL IN DEVELOPMENT !

## How to develop

### Requirements

- firebase realtime database credentials
- Node.js

### Steps

- copy `src/firebase-config.example.js` to `src/firebase-config.js` and fill values
- npm install
- npm start
